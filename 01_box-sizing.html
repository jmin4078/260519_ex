1. 박스 모델 및 박스 사이징 (box-sizing) 실습

개발자 도구(F12 -> Elements -> Computed)를 열어 두 박스의 실제 가로폭(width)을 측정해 보세요.

1) content-box (기본값)

CSS 지정 크기(200px) 외에 패딩(좌우 40px)과 보더(좌우 10px)가 합산되어 실제 가로 크기 250px로 불어납니다.

content-box 테스트

2) border-box (실무 표준)

패딩과 보더를 200px 안에 자동 흡수시켜 실제 가로 크기 200px를 그대로 유지합니다.
